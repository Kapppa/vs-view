# vsview-audio-convert

A [vsview](https://github.com/Jaded-Encoding-Thaumaturgy/vs-view) plugin for converting audio sample types and resampling audio clips.

## Requirements

- [vsview](https://github.com/Jaded-Encoding-Thaumaturgy/vs-view)
- At least one of the following VapourSynth plugins:
  - [ares](https://github.com/ropagr/VS-AudioResample) (recommended, supports resampling and quality settings)
  - [atools](https://github.com/ropagr/VS-AudioTools) (fallback, sample type conversion only)

## Installation

```bash
pip install vsview-audio-convert
```
