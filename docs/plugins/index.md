# Plugins

**VSView** features a plugin system that allows you to extend its functionality. Plugins can add new tools, custom UI panels, and integrate with the video and audio processing pipeline.

## Plugin Categories

Plugins are categorized based on their source and maintenance level:

- [**First-party (Official)**](first-party.md): Integrated directly into the **VSView** source code.
- [**Second-party**](second-party.md): Developed by community members in the official repository but maintained as separate packages.
- [**Third-party**](third-party.md): Created by independent developers in the community.

## Developing Plugins

See the [Plugin Development](development.md) guide to learn how to create and register your own **VSView** plugins.
